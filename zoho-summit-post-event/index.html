📌 Нотатки для запуску в SendPulse

Що зробити перед запуском

  • Деактивувати старий Welcome flow (з квитком). Новий Welcome flow для пост-евент аудиторії замінює його.
  • Якщо хочете зберегти старий flow як архів — у SendPulse можна вимкнути його (toggle), не видаляючи. Тоді ви матимете обидва на майбутні івенти.
  • Розсилка для зареєстрованих (attended + no_show) створюється як одна Кампанія на 28.05 з Filter element всередині — без дублювання логіки.

Дані користувача

  • Ім'я — береться автоматично з Telegram-профілю через змінну {{first_name}}. Окремо не запитуємо.
  • Email — не збираємо на етапі реєстрації в боті. Запитується пізніше, якщо користувач сам захоче зв'язатися (натисне «Записатись на консультацію»).
  • У Zoho CRM лід створюється з telegram_id як унікальним ідентифікатором, first_name і username (якщо є в Telegram).

Формат тайм-кодів

  • Усі тайм-коди в текстах — це HTML-посилання <a href="..."> на конкретний момент відео.
  • Формат у SendPulse: [13:19](https://youtu.be/lawo-3Ic90c?si=fLS1f9XA4cFTry-b&t=799s) — у месенджері виглядатиме як клікабельне «13:19».
  • При копіюванні через кнопку «Копіювати» HTML-теги <a> зберігаються — вставляйте текст у поле повідомлення SendPulse як є.

Тег registered_post_event

  • Окремий тег для тих, хто реєструється після саміту — щоб не плутати з тими, хто був на самому саміті.
  • Дозволяє у майбутньому окремо комунікувати з цією аудиторією (наприклад, запрошувати на наступний саміт).

Кнопка «Дивитись повний запис» (URL-button)

  • Це кнопка типу URL у SendPulse (не Quick Reply, не Continue flow).
  • URL для кнопки: https://youtu.be/lawo-3Ic90c?si=fLS1f9XA4cFTry-b
  • Назва кнопки: 🎬 Дивитись повний запис
  • У SendPulse: при додаванні кнопки в Повідомлення → тип «URL» → вставити посилання.

Кнопка «Записатись на консультацію»

  • Натискання кнопки → тег wants_consultation. Тут вже можна попросити email/телефон, якщо потрібен для зворотного зв'язку.
  • Окремий короткий flow для записи: запит контактних даних, передача в Calendly або Zoho Bookings.
  • Окремим завданням пізніше.
